##  Openeuicc 
This provide E-sim support on  E-sim supported android devices on gsi's.

##  How to use this file ?
   
    Clone this repo,
    copy openeuicc and openeuicc-deps to rom's -> packages/apps
   
    And add these lines in GSI Makefile:

    PRODUCT_PACKAGES += \
     OpenEUICC \
